# HTTP Auto Responder

Useful tool for static responses. It's like Fiddler's AutoResponder.

## Features

-   Wildcard Rule definition
-   Simple UI to manage Rules and Responses
-   Farx & Saz import

## Installation

```
go get -u github.com/asalih/http-auto-responder
```
Then edit config.json
